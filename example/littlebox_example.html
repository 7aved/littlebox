It's easy to include a littlebox in your project:

Step 1: Download the littlebox CSS files and initialize the plugin in your project (as well as Bootstrap, if not already initialized):
Download files

+

<link rel="stylesheet" type="text/css" href="lib/littlebox.min.css" />
<link rel="stylesheet" type="text/css" href="lib/bootstrap.min.css" />
Step 2: Add a littlebox to your code. The same structure is used for any littlebox:
<lilbox>
   <lildiv></lildiv>
   <lildiv></lildiv>
   <lildiv></lildiv>
</lilbox>
Step 3: Add a littlebox class to set an icon in your littlebox:
<lilbox class="soundbar-box">
   <lildiv></lildiv>
   <lildiv></lildiv>
   <lildiv></lildiv>
</lilbox>

Each littlebox icon is made up of three elements (<lildivs>) that are completely styled in css. Some littlebox icons are animated, others are not. Icons can be changed onEvent simply by switching and swapping classes with JavaScript.

These littleboxes work great as button groups and links:


class="check-box lb-green lb-group-left"
class="x-box lb-red lb-group-middle"
class="info-box lb-group-right lb-wide"
(place <lilbox> elements back-to-back or place comments between <lilbox> elements to fight small inter-box space seen here)



class="play-box lb-gray lb-group-left lb-group-top"
class="soundbar-box lb-group-left lb-group-top"
<br>
class="earbud-box lb-group-left lb-group-bottom"
class="pause-box lb-gray lb-group-right lb-group-bottom"


Simply wrap your <lilbox> link in an <a> tag like you would any other link.
